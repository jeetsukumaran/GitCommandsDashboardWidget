Searching

In all of the following, regexp may be replaced by more complex multi-component expressions using -e, --and, --or and/or --not arguments.

Searching for Content

git grep regexp
Search working tree for text matching regular expression regexp.
git grep regexp $(git rev-list --all)
Search working trees of all revisions for text matching regular expression regexp.
git grep regexp $(git rev-list rev1..rev2)
Search working trees of all revisions between rev1 and rev2 for text matching regular expression regexp.
