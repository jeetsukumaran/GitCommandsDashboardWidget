Searching

In all of the following, 'regexp' may be replaced by more complex expression using the -e, --and, --or and/or --not arguments.

Searching for Content

git grep 'regexp'
Search working tree for regular expression regexp.
git grep 'regexp' $(git rev-list --all)
Search working trees of all revisions for regular expression regexp.
git grep 'regexp' $(git rev-list rev1..rev2)
Search working trees of all revisions between rev1 and rev2 (which may be specified using a SHA1 commit ID, branch name, or tag) for regular expression regexp.
