Branching

git branch
List all local and remote branches.
git branch -r
List all local branches.
git branch new-brname
Create a new branch named new-brname, referencing the same point in history as the current branch.
git branch new-brname rev
Create a new branch named new-brname, referencing rev, which may be specified using a SHA1 commit ID, branch name, or tag.
git branch --track new-brname remote/remote-brname
Create a new tracking branch named new-brname, referencing, and pushing/pulling from, branch remote-brname, on remote repository remote.
git branch -d brname
Delete the local branch brname (fails if brname is not reachable from the current branch).
git branch -D brname
Force delete of the local branch brname (works even if brname is not reachable from the current branch).
git branch -d -r remote/remote-brname
Delete a "local remote" branch, i.e. a local tracking branch.
git push remote :heads/remote-brname
Delete a branch from a remote repository.
git checkout brname
Make the current branch brname, updating the working directory to reflect the version referenced by brname.
git checkout -b new-brname
Create a new branch new-brname, referencing the current point, and check it out (switch to it).
git checkout -b new-brname rev
Create a new branch new-brname, referencing rev, which may be specified using a SHA1 commit ID, branch name, or tag, and check it out (switch to it).
