Branching

Listing Branches

git branch
List all local branches.
git branch -r
List all local and remote branches.

Creating Branches

git branch new-branch
Create a new branch named new-branch, referencing the same point in history as the current branch.
git branch new-branch rev
Create a new branch named new-branch, referencing rev, which may be specified using a SHA1 commit ID, branch name, or tag.
git branch --track new-branch remote/remote-branch
Create a new tracking branch named new-branch, referencing, and pushing/pulling from, branch remote-branch, on remote repository remote.

Switching Branches

git checkout branch
Make the current branch branch, updating the working directory to reflect the version referenced by branch.
git checkout rev
Check out (switch to) version rev, without explicitly branching. Running "git checkout -b new-branch" will create a branch from the checked out version.

Simultaneous Creating and Switching Branches

git checkout -b new-branch
Create a new branch new-branch, referencing the current point, and check it out (switch to it).
git checkout -b new-branch rev
Create a new branch new-branch, referencing rev, which may be specified using a SHA1 commit ID, branch name, or tag, and check it out (switch to it).

Deleting Branches

git branch -d branch
Delete the local branch branch (fails if branch is not reachable from the current branch).
git branch -D branch
Force delete of the local branch branch (works even if branch is not reachable from the current branch).
git branch -d -r remote/remote-branch
Delete a "local remote" branch, i.e. a local tracking branch.
git push remote :heads/remote-branch
Delete a branch from a remote repository.
