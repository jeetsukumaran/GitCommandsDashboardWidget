Reverting

git revert rev
Reverse commit specified by rev, which can be any SHA1 commit ID, branch name, or tag, and commit the result. This does not do the same thing as similarly named commands in other VCS's such as "svn revert" or "bzr revert".
git checkout path(s)
Re-checkout file or files specified by path(s), overwriting any local changes. This is most similar to "svn revert".
git checkout rev path(s)
Re-checkout file or files specified by path(s) to version specified by rev, which can be any SHA1 commit ID, branch name, or ta, overwriting any local changes.
